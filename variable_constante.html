Les Variables/Constantes

Les Variables

Il existe plusieurs types pour les variables:
Il y a les variables de type numérique:
-INTEGER: Tous les nombres non décimaux
-REAL: Tous les nombres non-complexes

Et celle de type Caractères:
-CHAR: Un caractère unique
-STRING: Une chaîne de caractères

Enfin, il y a le type BOOLEAN, qui se caractérise par deux possibilités: VRAI ou FAUX


Une variable se déclare de cette façon dans un programme Pascal:
(var):(type)
Lors des instructions, on attribue une valeur à une variable par la commande:
(var1):=(valeur);

De plus, une variable peut prendre la valeur d'une autre variable:
(var1) := (var2);

Enfin, on peut réattribuer à une variable sa propre valeur, permettant ainsi de ne pas perdre ce qui lui était déjà attribué:
(var1):=(var1)+(valeur);


Les Constantes

Une constante est une valeur nommée comme une variable mais qui ne change jamais (d'où le nom de constante). Elle se déclare avant les variables de la façon suivante:
CONST

(const):(valeur);
