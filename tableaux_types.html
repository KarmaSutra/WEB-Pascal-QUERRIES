Les Tableaux

Quand il faut stocker un grand nombre de valeurs, il est plus facile de les placer dans un tableau que de les attibuer chacune à une variable différente, en prenant le risque de se perdre avec le nombre de variables. Un tableau se déclare de la façon suivante:

VAR

tab: ARRAY OF [min..max,min..max ] OF type

Pour utiliser un tableau, on l'appelle comme une variable à la seule différence qu'il faut indiquer le rang du tableau à utiliser:

tab[x]

On peut déclarer une deuxième dimension au tableau de la façon dont c'est montré dans l'exemple, il faudra alors indiquer le rang des deux dimensions pour utiliser le tableau:

tab[x,y]

Les Types

Voici comment déclarer un type (après déclaration des variables, avant le début du programme):

TYPE nom_type= record

sous_type1: type1

sous_type2: type2

END;

Pour utiliser un type dans le programme, il suffit de faire appel au type puis au sous-type de cette manière:

type.sous_type
