Les Structures Pascal

Voici la structure basique d'un programme Pascal:
PROGRAM (nom_programme);

VAR
(VAR 1):(TYPE);
(VAR 2):(TYPE);

BEGIN
INSTRUCTIONS
END.

Cette structure concerne uniquement les programme les plus basiques comprenant uniquement des variables, qui sont d'ailleurs le point que nous allons maintenant aborder.
